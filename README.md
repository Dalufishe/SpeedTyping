### 我們計劃在半年內做出一個打字練習/測速網站, 含有以下功能 -
- 檢測打字速度和錯誤率
- 可選擇文章或程式碼
- 文章或程式碼提供庫或自訂上傳等來源
- 練習/競賽/競速等功能
- 排行榜/積分系統
- 論壇/特論區系統
- 外掛/腳本偵測系統
